Philoi

Online Learning

Project description

• Background

Client: Major League Hacking

Date: Aug 2022 – Jan 2023

Type: Web Application

During the Summer Hackathons, we had a topic of developing an application on the theme of Friendship. So, I thought of developing a game inspired by the "two lies/one truth" game. In this game, participants will share a fact about themselves, and the host will start the game and display the fact on the screen. The other participants will then have to guess who the fact belongs to, using a multiple-choice option.

• Problem

Oftentimes, people feel uncomfortable meeting new friends and sharing personal information. As a result, they resort to small talk.

Solution:

My program, Philoi, aims to solve this problem by allowing a host to break the ice by sharing interesting facts about individuals. With Philoi, the host can have information about multiple individuals and can use it to facilitate conversations among them. This program can be used in various settings, including social events, networking events, and team-building activities, to name a few. By using Philoi, individuals can connect on a deeper level and build stronger relationships.
